# Montgomery County Traffic Collision Project
### Overview
This project analyzes over 195,000 traffic collision records from Montgomery County, using police-reported crash data.
The dataset contains detailed information on:

- Crash date & time

- Road & environmental conditions

- Driver behavior (distraction, impairment, fault)

- Vehicle information

- Injury severity

- GPS coordinates

- Speed limit & road type

The goal is to identify collision hotspots, dangerous driving behaviors, and environmental risk factors, and provide recommendations to improve roadway safety.

## Project Goals

- Identify high-risk locations, roads, and intersections

- Determine peak crash hours and peak days

- Analyze collisions by weather, speed limit, light conditions, and vehicle make

- Study behavior-specific crashes such as distracted driving and impaired driving

- Understand patterns in injury severity

- Provide data-driven safety insights
